Official implementation for our paper: **Semantic-Guided Fast Adversarial Training via Class Relationship Exploitation**. (ICMR'26)

## Environment settings

- OS: Ubuntu 20.04.3
- GPU: NVIDIA RTX 3090
- Cuda: 11.4
- Python: 3.8.10
- PyTorch: 1.10.1
- Torchvision: 0.11.2

## Execute command

```.bash
./main.sh  # for train
./test.sh  # for test
```
